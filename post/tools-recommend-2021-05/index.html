本文推荐的都是我正在用的或者用过的产品

硬件

Macbook Pro 13寸

如果是开发用，多配点内存会好些，磁盘一般256足够了

HHKB BT

我是被种草的，据说是程序员必备神器，用了三年了，按起来很舒服，偶尔用同事的薄膜键盘会很不适应。 一开始我是买的有线的，但是我觉得累赘。就在咸鱼上换了一个蓝牙版的，结果在办公室的多蓝牙设备的环境下，总是断连。 声音比机械的要轻，不脆，键程适中。

站立工作台

办公室必备啊，坐的时间长了腰会很不舒服，站起来可以调节高度。

trackpad2触控板

很多人不理解为什么macbook有触控板，还要单独买一个呢。一个是因为面积大，另外它的手感确实不错，其次有的时候笔记本是合盖的。

树莓派

这个嘛，就是玩具性质居多了。一开始心血来潮买了，折腾两天不知道干啥了。 然后搬家，租的房子里面没有有线电视，于是给我的树莓派安装了kodi系统，作为一个电视盒子。 后来买了投影仪，不需要电视了，就把树莓派装回debian，安装syncthing，作为nas中重要文件的备份机和镜像站。

投影仪

额，这个是最近才开始的，因为旅行中的一家民俗而种草。隔了一年多才拔草。 在卧室的白墙上，投影120吋的画面，家庭影院的既视感，舒服。

颂拓9运动手表

这个用了三年了，模式众多，跑步、游泳、滑雪、爬山等几十种。监测睡眠等也不在话下。

PC通用软件

Emacs

这个是和vim一样古老的编辑器，被誉为“神的编辑器”，也被成为“伪装成编辑器的操作系统”。 插件数量众多，质量都比较好。没有商业支持，都是用爱发电。 我自从16年接触Emacs，一开始用的spacemacs配置，入坑多次，退烧多次。去年又重新入坑，舍弃成熟的配置，自己从零开始。 目前比较稳定，目前这篇博客就是使用Emacs编写，使用ox-export插件导出为markdown，再由hugo编译成html静态网站。 编写代码和GTD也都是使用的Emacs。或者说能用Emacs干的事情我都在用Emacs。

Vim

和Emacs齐名的古老的编辑器，被誉为“编辑器之神”，单从用户数和用户熟知度来说，能完胜Emacs。 在服务器上面编辑文本文件，以及在mac上面编辑配置文件都是习惯性使用vim。 当然我并没有学vim的高级用法，就是最基本的移动和编辑功能。

Chrome

这个是程序员用的比较多的浏览器了，尤其是前端程序员。插件众多，开发工具箱比较强大。

Listen 1

一个开源的音乐项目，各个平台均匀客户端。尤以chrome的插件版体验最好，可以登录网抑云和qq音乐，播放自建的歌单。 它集成了4-6个平台的歌曲，均免费。

Bitwarden

开源的密码管理器，类似1password。

Trojan客户端

翻墙的工具。需要购买相应的服务。

Rime

开源的中文输入法，内置十几种输入方案，有拼音、双拼、五笔、郑码、笔画、文言文等。良心之作

Cyberduck

开源的FTP/Webdav客户端

LibreOffice

开源的office客户端，多平台支持。

Calibre

开源的图书管理系统，借助插件可以完成更多的事情。

VirtuxBox

开源的虚拟机软件

Wakatime

统计开发人员的时间消耗，例如编辑器比例、编程语言的比例等

VS Code

开源的IDE，不多说

RescueTime

统计设备的各个软件的使用时间。

Charles

代理工具，通常用来网络抓包

Syncthing

开源的p2p文件同步工具，跨平台

GIMP

开源的图片编辑器，PS的代替品

DesktopNaotu

百度脑图的离线版

digikam

图片管理工具

QBittorrent

bt下载工具

Anki

开源的记忆卡片工具

Mac软件

mac下的专属软件

Motrix

迅雷的代替品

MAMP

LAMP的代替品，快速启动php+nginx+mysql服务

MoneyWiz

记账工具，收费，免费版无同步功能

MarginNote

收费软件，pdf及各种文档的阅读&标记工具

QuickSliver

mac下的快速启动工具，开源

amethyst

窗口布局快速排版工具

Iterm2

最好用的终端工具

Android软件

AnkiDroid

Anki的手机版

APKPure

可以代替Google 应用商店的软件，也需要翻墙

Bitwarden

Bitwarden的手机端

Clash

Trojan的手机端

DAVx

CardDav和CalDav的同步工具，用来同步通讯录和日历

F-Droid

开源的安卓软件市场，只有开源软件。

Kiwi Browser

基于chrome的手机浏览器，可以安装chrome插件

KOReader

手机端的pdf文档阅读工具

Listen 1

Listen 1的手机端，没有电脑端的功能多

Orgzly

Org-mode的手机编辑器

RescueTime

记录手机APP的使用时间

Syncthing

Syncthing的手机端，可与电脑端同步数据

书迷

用来看网络小说，无广告，源多

Termux

安卓下的终端软件，模拟了alpha系统，可以使用apk 命令安装各种软件

Chrome插件

RSSHub Radar

发现并订阅当前页面的rss链接

FeHelper

前端开发必备工具面，json美化、时间戳转化等常用小工具

BitWarden

bitwarden的网页密码填充插件

floccus

使用webdav来备份/同步浏览器书签

Adblock Plus

拦截网页广告

PT Plugin Plus

玩PT圈子的必备工具，一个页面管理众多PT站点

WakaTime

用来记录chrome在开发中的时间花费

ResuceTime

用来记录页面浏览中的时间花费类别

Emacs插件

straight

包管理工具，如果无特殊要求的话，会觉得它好麻烦，但是如果对某个插件进行了自定义的话，会发现这个超爽

use-package

把关于某个插件的配置都集中到一起。且管理了package的依赖顺序、加载时机等，配合straight食用更加

ag

用来代码搜索

rime

输入法

phi-search

替换默认的i-search,解决rime带来的不能搜索中文的问题

doom-themes

比较极客的主题风格

all-the-icons

好康的图标不能少

super-save

自动保存插件

amx

快速输入emacs命令

avy

屏幕可视区域内的快速光标定位

anzu

i-search结果的统计

helm

智能补全前端

doom-modeline

状态栏美化

linum

显示行号

wakatime-mode

wakatime的emacs插件

keyfreq

记录所有的按键频率

alert

桌面级通知

ox-hugo

把org导出为markdown

editorconfig

集成.editorconfig

lsp-mode

lsp的eamcs前端

expand-region

代码块的快速选择

ivy

另一个智能补全前端

company

另另一个代码补全前端

rainbow-delimiters

为嵌套的括号分配不同的颜色

so-long

打开大文件的时候，关闭所有占用资源的模式

hideshow

展开/收起代码块

ranger

ranger的emacs前端

treemacs

一款文件树侧边栏

projectile

快速处理项目目录的相关命令

magit

最好用的git客户端

org

用来写文字、GTD等强大的工具

org-roam

org的增强，注重文件中间的关联关系

org-superstar

org文件中的星号美化

org-pomodoro

org在GTD过程中的番茄钟工具

hydra

自定义键位方案

meow

有别于evil的多模式快捷键方案

Cli

zsh && oh-my-zsh

配合iterm2，谁用谁知道

ranger

文件夹导航工具

nix

包管理工具，目前我逐渐使用nix代替mac的brew

git

不解释

pandoc

多种文本格式的文档互相转换

tldr

too long don’t read 简化版的man手册

其它

小鹤双拼

用双拼输入中文真的很快，（前提是花两个月的时间来熟悉它）
